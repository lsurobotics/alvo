Tutorial 3: Editing and Re-using Tasks

Task:

  1. Move the yellow box from the left side of Station A to the right side and turn it upside-down.
  2. Move the blue box from the left side of Station B to the right side and turn it upside-down.
  3. Move the orange box from the left side of Station C to the right side, on top of the green box, and turn it upside-down.

As you have seen in the previous tutorial, robot programs are often very similar to each other. It can therefore be useful to use existing code rather than writing all programs from scratch.

The goal of this tutorial is similar to the previous one, and you could solve it like you did before, but we will use a different method to program this time. You can click on each step to get more detailed instructions.

  1. Re-use the first task
    1. Open the menu below.
    2. Instead of creating a new task, drag the existing task onto the canvas and attach it to the bottom of the program.
    3. You should now have two almost identical task blocks right underneath each other. As you can see, both are highlighted at once. Assign the new block to "Station B" using the menu.
    4. Before you continue, take a moment to understand what the current program does. Click the button underneath the simulator to try it out. As you can see, the program executes the exact same commands first at station A and then at station B.
  2. Turn the box upside-down at both stations
    1. Open the menu in the task editor and drag the block onto the canvas. The robot can only turn blocks that it is currently holding, so the new block has to be placed right after the block.
    2. As soon as you place the new block, a warning should appear. It asks you whether you want to edit all places where the current task is used, or create a copy first. For now, click on the button.
    3. Drag another block from the menu onto the canvas and place it right after the one you just added.
    4. Before you continue, take another moment to understand what the current program does and try it out by clicking the button underneath the simulator. As you can see, the program still executes the exact same commands first at station A and then at station B. This is because you selected the button and your edits have been applied to the programs at station A and B at once.
  3. Re-use the first task again
    1. Since the third part of the task is also similar to the previous ones, you can use the existing code as a starting point instead of creating a new task. Open the menu below, drag the existing task onto the canvas and attach it to the bottom of the program as you did before.
    2. You should now have three almost identical task blocks right underneath each other, all of which are highlighted. Assign the new block to "Station C" using the menu.
  4. Change the locations for station C
    1. Make sure you have selected the program for station C in the left canvas.
    2. Open the menu and select . Select the middle left location and drag it onto the canvas.
    3. As soon as you place the new block, the same warning as before should appear. This time, click on the button since you only want to change the program for station C.
    4. Enter a new name for the task for station C, for example "Move stacked box to the right". As you can see, the left canvas has changed now. The currently selected task has been renamed and is the only one highlighted. You are now only editing this task.
    5. Continue editing the current task: attach the "middle left" location you just created to the block, replacing the "bottom left" block. You can delete the leftover "bottom left" block by dragging it onto the trash can symbol if you want.
    6. Repeat the previous steps and define another new location at the middle right. Attach it to the block, replacing the "bottom right" block. Again, you can delete the leftover block if you want.
    7. Test your solution by clicking the button below. Watch how the robot executes each step of the program. Notice how the programs for the stations A and B have not changed after you created a copy of the task.
