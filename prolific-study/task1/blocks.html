Task 1: Item Factory

Task:

This task is controlled by the two buttons at the bottom of the simulator.

  1. When Button 1 is pressed, the robot should move the blue box from Station A to the item processor machine at Station C.
  2. When Button 2 is pressed, the robot should move the orange box from Station B to the item processor machine at Station C.
  3. After processing of either item is complete, the robot should move them into the chute at Station C. The order in which they get placed into the chute does not matter.

Note: If you want to try out your program manually, we recommend using the Run Program button next to the simulator (where you can manually click the red and blue button). When you click on the Test Current Program button below, the manual buttons will be disabled and the simulator will trigger them for you.
