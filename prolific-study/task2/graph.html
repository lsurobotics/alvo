Task 2: Working in Parallel

Task:

For this task, the robot has already been programmed to work on the stations on the left side (Stations A, B and C). The existing code cannot be changed. We recommend that you try out the given program before starting the task. Note that at the beginning of this task, Station B has just been used and needs to cool down before the robot can load it. The given code already considers this and works as intended.

Your task is to add the following functionality that is controlled by the two buttons at the bottom of the simulator:

  1. When either of the two buttons (Button 1 or Button 2) is pressed, the robot should follow the same workflow as on the left: It should move the blue box from Station D to the item processor machine at Station E, and to Station F once the item has been processed.
  2. Both of the two workflows (the existing one and the part programmed by you) need finish for the task to be completed successfully. Ideally, they should be interleaved (with the robot doing something for the other task while it is waiting), but the only mandatory requirement is that they don't interfere with or block each other

Note: Since your task is similar to the given one, you might want to re-use the existing blocks if you want. If you right-click on one, you can select "Clone" to make a copy that you can edit.
